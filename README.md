# VsCode ShortCuts
> Daily Useful
> * `Ctrl + B` = Explorer Open/Close
> * `Ctrl + J` = Terminal Open/Close
> * `Ctrl + \` = Spilt Open
> * `Ctrl + f4` = Spilt Close
> * `Ctrl + N` = New File
> * `Ctrl + Tab` = Go to Previous File
> * `f5` = Start Debug
> * `Shift + f5` = Stop Debug
> * `Ctrl + Alt + N` = Run Code
> * `Alt + Arrow Up/Downl` = Line Up or Down
> * `Ctrl + Alt` = Multi Vairable Create
> * `Ctrl + F` = Find
> * `Ctrl + H` = Replace
